## Key concepts
- **Arc testnet** (Chain ID 5042002) — USDC (6 decimals) is the native gas token
- **Vault system**: VaultFactory deploys non-custodial Vault instances; SyntheticVault for synthetic assets
- **AMM**: AMMPool + AMMRouter for synthetic asset trading
- **Registry pattern**: AssetRegistry, StrategyRegistry, ReasoningTraceRegistry for on-chain provenance
- **Price oracle**: PriceOracle with Chainlink integration path
- **Foundry tests**: Use `forge-std/Test.sol`, `vm.prank`, event assertions
- **MockUSDC**: ERC20 with `decimals() = 6` for test fixtures

- ABIs in `contracts/abis/` are committed (generated by `forge build`)
- Two-eyes review required on any contract change (Bogdan / Dan)
